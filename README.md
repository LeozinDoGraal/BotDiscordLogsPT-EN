# BotDiscordLogsPT-EN

# 🔒 Discord Advanced Logs Bot

Sistema avançado de logs administrativos para Discord.

---

# 🇧🇷 Português

## ⚙️ Configuração

### 1. Instale as dependências

```bash
pip install -U discord.py
```

### 2. Configure o token

```py
TOKEN = "SEU_TOKEN_AQUI"
```

### 3. Ative os intents no Discord Developer Portal

- Presence Intent
- Server Members Intent
- Message Content Intent

### 4. Execute o bot

```bash
python bot.py
```

### 5. Crie os canais de logs

```bash
.criarlogs
```

# 🇺🇸 English

## ⚙️ Setup

### 1. Install dependencies

```bash
pip install -U discord.py
```

### 2. Configure the token

```py
TOKEN = "YOUR_BOT_TOKEN"
```

### 3. Enable intents in the Discord Developer Portal

- Presence Intent
- Server Members Intent
- Message Content Intent

### 4. Run the bot

```bash
python bot.py
```

### 5. Create the log channels

```bash
.createlogs
```
